This is the idea of the project

The main pourpose of this project is to make a web app that contains the information of various rocket launches, some historical ones, some that just happened, a countdown for the next rocket launch and some information about the next launches for the month, it could be displayed by time, by the company that operate the rocket, by the payload among others.

Project description

I think this is a very good project not just because the wonderfull space pictures, I think it's great because of the scalability of it too, we could make a simple page to begin with, exploting an api, like the one that gives you information about the spaceX launches and then we could make our own database of rocket launches, we have the javascript countdown and much more, i think that this would be a nice project.
